# 📖 HTML & CSS Interactive Learning Book

A self-contained, multi-chapter interactive learning resource for HTML and CSS — built entirely with HTML, CSS, and Font Awesome icons. Designed as a personal study reference and portfolio project by **Najmun Naher**.

---

## ✨ Features

- 📚 Chapter-by-chapter breakdown of HTML & CSS concepts
- 🧭 Table of Contents sidebar on every chapter page for quick navigation
- ⬅️ ➡️ Previous / Next navigation buttons between chapters
- 🏥 Real-world project: a **Hospital Management System** homepage (`index.html`)
- 📱 Responsive layouts demonstrated with live CSS examples
- 🎨 Live CSS demos embedded directly inside chapter pages (transitions, animations, cascading, backgrounds, etc.)
- 🔗 Central `book.html` hub linking all chapters

## 🏥 Real-World Demo: Hospital Management System

`index.html` is a complete hospital homepage demonstrating applied HTML & CSS skills:

- Sticky header with logo, hospital name, and contact info
- Navigation bar with social media icon links
- Hero banner section with call-to-action
- Services section (3-column layout)
- Notice board and featured video panel
- News & Events, Medical Services list, Photo Gallery
- Footer with copyright

## 🚀 Getting Started

No build tools or server needed — this is pure HTML/CSS.

```bash
# 1. Clone the repository
git clone https://github.com/your-username/html-css-book.git

# 2. Open in your browser
open index.html        # Hospital homepage
open book.html         # Chapter hub
```

Or simply double-click any `.html` file to open it in your browser.
